OBJECTIVE: To create a calculator using REACT

Tools:REACT, JSX

WORKING  METHOD:  The project works on using three components inside the APP.JS: Screen, display and numbers.
                -> Number and scree are responsible for handling numbers and display if responsible for handling the output

Data Structure used: For simplicity, Stacks was used to build this calculator
